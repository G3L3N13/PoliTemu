# 📚 Plataforma Inteligente de Intercambio Académico
## PoliTemu
Plataforma web para compra y venta de artículos o servicios dentro de la Escuela Politécnica Nacional

</div>

---

## 📌 Descripción del proyecto
Este proyecto consiste en el desarrollo de una plataforma web inteligente diseñada para los estudiantes de la Escuela Politécnica Nacional, cuyo objetivo es facilitar el intercambio de bienes y servicios académicos entre alumnos.De esta manera, la plataforma facilitará que los usuarios puedan publicar, buscar e intercambiar recursos académicos como apuntes, libros, tutorías o servicios relacionados al ámbito universitario en una sola plataforma centralizada, fomentando la colaboración y seguridad dentro de la comunidad estudiantil.

---

## ❗ Problema que aborda

Actualmente, los estudiantes no cuentan con una plataforma centralizada para intercambiar recursos académicos, lo que genera:

- Pérdida de información útil (artículos, libros, etc.)
- Pérdida de tiempo cuando los artículos ya han sido vendidos
- Uso de múltiples canales desorganizados en Whatsapp
- Falta de seguridad y confianza en los intercambios

--- 

## 🎯 Objetivo
Desarrollar un sistema web que centralice el intercambio académico de forma organizada, segura y eficiente de recursos entre estudiantes de la Escuela Politécnica Nacional.
