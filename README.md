# PoliTemu
Plataforma web para compra y venta de artículos o servicios dentro de la Escuela Politécnica Nacional

# Tecnologias utilizadas
- Backend: React + Vite
- Frontend: HTML, CSS, JavaScript
- Servidor: Node.js
- Control de verisiones: Git/GitHub
